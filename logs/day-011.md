## Learned

**Text manipulation commands:**

| Command | What it does |
|---------|-------------|
| `cut -c 1-5 file.txt` | Extracts characters 1 to 5 from each line |
| `cut -d ',' -f 1 file.txt` | Cuts by delimiter, extracts field 1 |
| `paste file1.txt file2.txt` | Merges two files side by side |
| `paste -d ',' file1.txt file2.txt` | Merges files with custom delimiter between them |
| `head file.txt` | Shows first 10 lines of a file |
| `head -n 5 file.txt` | Shows first 5 lines specifically |

**Tab character fix:**

| Method | How to use |
|--------|-----------|
| `\t` with echo -e | `echo -e "word\tword"` inserts real tab |
| Tab key in nano | Works normally with no workarounds needed |

Key insight on `head` - extremely useful for large files
where you only need a quick look at the beginning without
loading the entire content. Pairs well with pipes:

```bash
cat largefile.txt | head -n 20
```
