Common combinations:
- `755` - standard for executables and directories
- `644` - standard for regular files
- `700` - private, owner only
- `777` - everyone full access, avoid this

**Ownership commands:**

| Command | What it does |
|---------|-------------|
| `chown user file` | Changes file owner |
| `chown user:group file` | Changes owner and group at once |
| `chgrp group file` | Changes group only |

Note - both `chown` and `chgrp` require `sudo` because
changing ownership is a privileged operation.
