## Learned

**Core concept:**
Bash scripting is programming using the terminal.
Everything learned so far - commands, pipes, redirects,
permissions - combined with programming logic to
automate tasks.

Every bash command follows the same structure:

```bash
command argument1 argument2 argument3
```

**Quoting rules:**

| Syntax | What it does |
|--------|-------------|
| `echo Hello World` | Two separate arguments |
| `echo "Hello World"` | One single argument |
| `echo "$variable"` | Double quotes allow variable expansion |
| `echo '$variable'` | Single quotes treat everything literally |

**First script structure:**

```bash
#!/bin/bash
echo "Hello World"
```

The `#!/bin/bash` shebang at the top tells the system
which interpreter to use. Every bash script starts
with this line.

**Making a script executable and running it:**

```bash
chmod +x script.sh
./script.sh
```
