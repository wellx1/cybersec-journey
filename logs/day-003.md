## Learned

**Navigation:**

| Command | What it does |
|---------|-------------|
| `cd /foldername` | Go to that specific folder |
| `cd .` | Current directory (where you already are) |
| `cd ..` | Move up one level to parent directory |
| `cd ~` | Go home — shortcut for /home/kali |
| `cd -` | Jump back to previous directory |

**Listing files:**

| Command | What it does |
|---------|-------------|
| `ls` | List files in current directory |
| `ls -a` | Show hidden files too (dot prefix files) |
| `ls -l` | Detailed view — permissions, size, date |
| `ls -r` | Reverse alphabetical order |
| `ls -la` | Detailed view AND hidden files combined |

**File operations:**

| Command | What it does |
|---------|-------------|
| `touch filename.txt` | Create an empty file |
| `rm filename.txt` | Delete a file permanently — no recycle bin |

**Key insight:** Flags can be combined for more precise output.
`ls -la` gives you everything at once — hidden files with full
details. This is the version used most in real security work.

**Important warning learned:** `rm` is permanent and
irreversible. Always double check before running it.
Never run `rm -rf /` — it deletes everything on the system.
